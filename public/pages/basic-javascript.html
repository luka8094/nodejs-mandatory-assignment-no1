Variables in Javascript


Before we venture into NodeJS lets discuss some fundamental Javascript first.


In Javascript syntax its possibe to declare value through three different keywords: var, let, and const.

Its important to note that they all have difference means and purpose. Its common, if not best, practice to avoid var keyword and use let and const in most case.

One of the reason is due the scope of var keyword which could lead to conflicts and difficulties in code maintainability.

let has a block specific local scope which avoids this issue.
