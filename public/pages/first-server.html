What is NodeJS?

NodeJS is a runtime environment built around Javascript. The environment is wrapped around Google Chrome's V8 engine (browser engine) from which the web application is executed.

An important emphasis needs to be put on that its not a framework which is a common misunderstanding. NodeJS itself is open source and executable on multiple environments. It serves for various purposes such as building web or mobile solutions.

Some notable examples of high-profile businesses which utilize NodeJS for their services are Uber and Paypal.
